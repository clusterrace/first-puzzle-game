# Project: First Puzzle Game (Godot)

## Git Workflow — MANDATORY

- **Always push to origin after committing.** Every agent must run `git push origin main` after creating commits. Do not leave commits unpushed.
- Before starting work, run `git pull origin main` to ensure you have the latest changes.
- Commit messages should reference the KAMA issue identifier (e.g., `KAMA-XX`).
